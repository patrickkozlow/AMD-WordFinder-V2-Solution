File Overview:

wordfinder_v2 - original files

wordfinder_v2_solution.py - python script that calls original files to solve problem
import_data.py - supporting functions to import files 

test_wordfinder_v2_solution.py - unit tests for solution script
test_import_data.py - unit tests for import script

----------------------------------------------------------------------------

Running Instructions (tested with windows + Python 3.9.1):

If not installed, download and install up to date python distribution
Link: https://www.python.org/downloads/

1. Download and extract file to directory
2. Open and run with prefered IDE (VsCode, PyCharm, ect..)

OR

2. Copy directory path eg. (C:\Users\Documents\Solution)
3. Open command prompt -> type cmd into the search bar
4. navigate to solution path -> type: cd <copied path>
5. to run the solution script type: python wordfinder_v2_solution.py
6. to run the unit test scripts type: python test_import_data.py
	OR: python test_wordfinder_v2_solution.py
